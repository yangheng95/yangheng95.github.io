OmniGenBench

A Comprehensive Benchmarking Platform for Genomic Foundation Models

Back to Projects

Abstract

OmniGenBench is a comprehensive benchmarking platform designed to evaluate and compare genomic foundation models across diverse tasks and datasets. As genomic foundation models become increasingly important in computational biology, there is a critical need for standardized evaluation frameworks that can assess their performance, generalizability, and practical utility.

This platform addresses the challenge of heterogeneous evaluation practices in genomic AI by providing a unified framework for benchmarking models on tasks including genome sequence classification, gene expression prediction, variant effect prediction, and more. OmniGenBench enables researchers to systematically compare different models and identify the most suitable approaches for specific applications.

By providing standardized datasets, evaluation metrics, and a flexible API, OmniGenBench facilitates reproducible research and accelerates the development of next-generation genomic foundation models.

Supported Tasks

  • Genome Sequence Classification: Classify genomic sequences by function or type
  • Gene Expression Prediction: Predict gene expression levels from sequence
  • Variant Effect Prediction: Assess the functional impact of genetic variants
  • Regulatory Element Identification: Identify promoters, enhancers, and other regulatory regions
  • Chromatin Accessibility Prediction: Predict open chromatin regions
  • Protein-DNA Binding: Predict transcription factor binding sites

Getting Started

# Clone the repository
git clone https://github.com/yangheng95/OmniGenBench.git
cd OmniGenBench

# Install dependencies
pip install -r requirements.txt

# Run a benchmark
python benchmark.py --model your_model --task classification

Applications

  • Model Selection: Compare foundation models to choose the best for your application
  • Model Development: Evaluate new models during development
  • Research: Conduct systematic studies on model capabilities and limitations
  • Education: Learn about genomic AI through hands-on benchmarking
