Open-source Projects

OmniGenomeBench

OmniGenomeBench is a large-scale in-silico benchmarking framework for genomic foundation models (GFMs). It addresses the lack of standardized tools for evaluating GFMs, which are essential for understanding and synthesizing genome sequences. The framework integrates millions of genomic sequences across a variety of tasks, automating the benchmarking process for diverse GFMs. OmniGenomeBench democratizes genome modeling by providing an open-source, user-friendly interface that supports complex tasks like RNA design and structure prediction, with a public leaderboard to track performance across models.

PyABSA

PyABSA (Aspect-Based Sentiment Analysis) is a modularized framework designed to simplify the implementation of sentiment analysis tasks. It offers pre-trained models and datasets for both research and production environments, making it easy to reproduce and benchmark ABSA tasks, including aspect term extraction and sentiment classification. PyABSA integrates with widely used machine learning libraries such as Hugging Face, enabling developers to deploy state-of-the-art ABSA models with ease. It is a robust tool for anyone looking to apply sentiment analysis in real-world applications, from research to enterprise use.
