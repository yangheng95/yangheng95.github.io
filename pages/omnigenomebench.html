OmniGenomeBench

Abstract

The advancements in artificial intelligence, such as Large Language Models (LLMs), have fueled expectations for breakthroughs in Genomic Foundation Models (GFMs). The genomic "code of nature," hidden in diverse genomes since the beginning of lifeâ€™s evolution, holds immense potential for impacting humans and ecosystems through genome modeling. Recent breakthroughs in GFMs, such as Evo, have attracted significant investment and attention, as they address long-standing challenges in genomic modeling. These models aim to transform in-silico genomic studies into automated, reliable, and efficient paradigms.

However, GFM studies face two major challenges: the lack of GFM benchmarking tools and the absence of open-source software for diverse genomics. These challenges hinder the rapid evolution of GFMs and their wide application in tasks such as understanding and synthesizing genomes, problems that have persisted for decades.

To address these challenges, we introduce OmniGenBench, a framework dedicated to GFM-oriented benchmarking. OmniGenBench standardizes benchmark suites and automates benchmarking for a wide range of open-source GFMs. It integrates millions of genomic sequences across hundreds of genomic tasks from four large-scale benchmarks, democratizing GFMs for a wide range of in-silico genomic applications.

Additionally, OmniGenBench is open-source, offering user-friendly interfaces and tutorials, applicable for AutoBench and complex tasks like RNA design and structure prediction. We also launched a public leaderboard showcasing benchmark performance derived from AutoBench. OmniGenBench represents a step toward standardizing GFM benchmarking and democratizing GFM applications.

Overview

OmniGenBench is an open-source benchmarking software platform for Genomic Foundation Models (GFMs). Fig. 1 presents the architecture of the platform, showcasing its extensive toolkit and genomic tasks.

Key Features

  • Open-source Genomic Software: Includes AutoBench, embedding tools, RNA design, augmentation tools, and structure prediction, all under an open-source model.
  • In-silico Genomic Tasks: Supports tasks such as mutation detection, structure prediction, mRNA design, and phenotype classification.
  • Built-in Application Interfaces: Provides user-friendly wrappers and utilities to make working with GFMs more accessible for both researchers and developers.

Benchmarking Suites

The framework introduces four standardized benchmarking suites:

  • RNA Genomic Benchmark (RGB): Focuses on SN-level genome understanding tasks, such as predicting mRNA degradation rates.
  • Plant Genomic Benchmark (PGB): Evaluates GFMs specialized in plant biology through tasks like promoter strength prediction and gene expression regression.
  • Genomic Understanding Evaluation (GUE): A multi-species genome classification benchmark including tasks like genome sequence classification and gene expression prediction.
  • Genomic Benchmarks (GB): A collection of DNA genome datasets focusing on evaluating genomic sequence classification models.

Installation

To install OmniGenBench, use the following pip command:

pip install omnigenome -U

Or clone the repository from GitHub:

git clone https://github.com/yangheng95/OmniGenomeBench
cd OmniGenomeBench
pip install -e .

Usage Example

Here's how you can use the AutoBench class to automate benchmarking:

from omnigenome import AutoBench
gfm = 'LongSafari/hyenadna-medium-160k-seqlen-hf'
bench_root = "RGB"
bench = AutoBench(bench_root=bench_root, model_name_or_path=gfm, overwrite=False)
bench.run(autocast=False, batch_size=32, seeds=[42])

Conclusion

OmniGenBench is a powerful, flexible platform designed to address the challenges of genomic foundation model benchmarking. It provides standardized benchmarking suites, open-source software, and tools that democratize access to genomic tasks such as RNA design and structure prediction. Whether you're a researcher or developer, OmniGenBench offers an easy way to test and fine-tune models.
