OmniGenBench is a comprehensive automated framework for benchmarking genomic foundation models. As artificial intelligence transforms genomics, OmniGenBench provides open-source infrastructure to evaluate, compare, and advance these models. Supporting applications from RNA therapeutics to precision medicine, our framework enables reproducible evaluation across the entire field of computational genomics.

Quick Start

Installation

pip install omnigenome

Basic Usage - Model Evaluation

from omnigenome import OmniGenBench

# Initialize benchmark
bench = OmniGenBench()

# Load a genomic foundation model
model = bench.load_model('OmniGenome-186M')

# Run AutoBench on multiple tasks
results = bench.auto_bench(
    model=model,
    tasks=['rna_secondary_structure', 'promoter_prediction', 'splice_site_prediction'],
    datasets='all'
)

# View results
bench.display_results(results)
bench.save_to_leaderboard(results)

RNA Design Example

from omnigenome.tasks import RNADesign

# Initialize RNA design task
rna_designer = RNADesign(model='OmniGenome-186M')

# Design RNA sequence for target structure
target_structure = '((((....))))'
designed_sequence = rna_designer.design(
    target_structure=target_structure,
    optimization_method='GRPO',  # Genetic Reinforcement Policy Optimization
    num_iterations=1000
)

print(f"Target: {target_structure}")
print(f"Designed: {designed_sequence}")
print(f"MFE: {rna_designer.evaluate_mfe(designed_sequence)}")

Custom Model Benchmarking

from omnigenome import OmniGenBench
from your_model import YourGenomicModel

# Initialize your custom model
your_model = YourGenomicModel()

# Wrap it for OmniGenBench
bench = OmniGenBench()
wrapped_model = bench.wrap_model(your_model)

# Run comprehensive evaluation
results = bench.evaluate(
    model=wrapped_model,
    benchmark='genomic_benchmarks',
    save_predictions=True,
    output_dir='./results'
)

# Compare with leaderboard
bench.compare_with_leaderboard(results)

üß¨ OmniGenome Foundation Model

State-of-the-art genomic foundation model introduced alongside OmniGenBench to tackle sequence sparsity and Single Nucleotide Variation (SNV) challenges. Delivers consistent RNA structure prediction performance across the benchmark suites documented in arXiv:2505.14402.

Key Achievement: SNV-aware pretraining pipeline detailed in the OmniGenBench publication

üöÄ GRPO Algorithm

Genetic Reinforcement Policy Optimization (GRPO) combines genetic search with reinforcement learning feedback to automate RNA sequence design loops within OmniGenBench. Captures the exploration and exploitation balance needed for reproducible RNA design experiments.

Innovation: Hybrid genetic + RL optimization published with the OmniGenBench toolkit

Making genomic AI accessible to all - OmniGenBench provides an open-source framework with CLI tooling, documentation, and leaderboard integration so research groups can evaluate genomic foundation models under a consistent protocol.

Impact: Open API supports custom datasets, models, and reporting workflows
